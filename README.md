# react-native-neo-airplay

Airplay for React Native

## Installation

```sh
npm install react-native-neo-airplay
```

## Usage

```js
import NeoAirplay from 'react-native-neo-airplay';

// ...

<NeoAirplay
  style={styles.box}
  tintColor="red"
  activeTintColor="green"
  prioritizesVideoDevices={true}
/>
```
